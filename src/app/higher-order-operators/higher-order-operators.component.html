Higher Order Operators

Overcome the multiple subscription (subscribe inside subscribe) - to avoid this we can use this higher order operators. we can easily find this operators which ends in map

    mergeMap
    switchMap
    concatMap
    exhaustMap



mergeMap- Operator

Example - this operator will subscribe and extract data and gives consolidated result.
Note: this will emits the data which is completed first , not in the order wise

Example - this is similar to mapTo , it will emits data based on source observable
